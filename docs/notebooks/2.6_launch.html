Using nf-core launch

Learning objectives
  • Use the nf-core launch web GUI to adjust parameters
  • Understand different options to using launch and when each would be applicable
  • Run a workflow in an offline mode

Construct an execution command

Open the nf-core launch website and select the rnaseq pipeline from the select a pipeline menu and select pipeline release 3.11.1. Then select ðŸš€ Launch.

On the far right, unhide Show hidden params and then fill out the Nextflow command-line flags section:

  • Leave -name blank. Doing this will ensure a random name will be applied, like it has been for our CLI runs so far
  • For -profile, enter singularity,c2r8
  • Leave -work-dir as default
  • Toggle -resume to true

Next, fill out the input section:

  • Copy and paste the full path to the samplesheet.csv
  • For -outdir, specify Exercise6
  • Enter your email address
  • Leave the other sections blank

Challenge

Now scroll through the remaining options, and fill in all of the parameters we have applied via our params file (including the multiqc_yaml file!)

Solution

Once you are satisified that the params are all accounted for (remember that the extra_args.config should have the quality decreased from 40 to 30), click Launch workflow
