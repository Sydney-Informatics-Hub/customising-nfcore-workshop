Nextflow tips and tricks

Query specific pipeline executions

nextflow log <run_name> -f

Run Nextflow in the background

The Nextflow config command is useful for understanding the resolved profiles and parameters that Nextflow will use run a pipeline.

nextflow run nf-core-rnaseq-3.11.1/workflow/main.nf -bg > workshop_tip.log

Capture a Nextflow pipeline’s configuration

nextflow config nf-core-rnaseq-3.11.1/workflow

Access Nextflow pipelines in private GitHub repositories

To interact with private repositories on GitHub or when running into GitHub API rate limits for public repositories (i.e. when using Nextflow Tower), you can provide Nextflow with access to GitHub by specifying your GitHub user name and a Personal Access Token in the scm configuration file inside your specified .nextflow/ directory.

providers {

  github {
    user = 'georgiesamaha'
    password = 'my-personal-access-token'
  }

}
